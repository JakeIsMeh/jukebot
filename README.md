# jukebot

### llm disclosure

unfortunately this was basically entirely written by LLMs (3.5 Flash thinking and a smidge of Sonnet 4.6), but I guided
the LLM throughout with very specific directions

### background

I wanted to write a music bot, which i eventually did, but the more important part of this repo is the src/framework
folder, which contains what I believe to be a very nice declarative type-safe framework for building command-based
discord bots.
